# GEMINI.md - Project Context & Orchestration Rules

## Project Overview: EngineAI Dashboard ("Self-Operating Agency")

The **EngineAI Dashboard** is the autonomous operational nervous system for Engine AI. It is designed to transform business operations from manual founder-led tasks to autonomous founder-led orchestration via a "Digital Assembly Line" of specialized hierarchical AI agents.

This project serves as:
1.  **Internal Engine:** Automating Engine AI's portfolio including BIAB (Business in a Box), Special Projects (SkunkWorks), Modular Builds (OpenClaw), and Desktop AI Setups.
2.  **White-Label Blueprint:** A rapidly deployable "Business in a Box" template for SME clients featuring autonomous repo cloning and AST-based refactoring.

### Core Strategic Focus
- **Decoupled Intelligence:** Oversight provided by Next.js UI; autonomous logic provided by a Hierarchical Agent Suite (Executive, Managerial, Specialist tiers).
- **Industrial Automation:** AST-based code refactoring and autonomous PR management for 24-hour client delivery.
- **Managerial Transparency:** Total visibility into agent reasoning and "thought loops" to eliminate human "blindness."
- **Intelligence Hub:** A bleeding-edge knowledge centre for founders to curate industry research and incubate new BMB modules.

## Technical Stack & Frameworks

- **Frontend:** Next.js 16 (React, Tailwind CSS).
- **Orchestration:** **Vercel AI SDK v6** (Agent class, built-in tool loops, MCP support).
- **Durable Logic:** **Vercel Workflows** for long-running (24h+) self-healing pipelines.
- **Backend/Database:** Supabase (PostgreSQL with RLS, durable memory).
- **Engineering:** AST-based refactoring for autonomous "Fill in the Blanks" repository adaptation.
- **Communication:** Telegram, Discord, Google Drive, Gmail/Calendar (via MCP).
